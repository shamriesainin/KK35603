Steps

1) Extract and copy the mininursery folder in your C:\xampp\htdocs or C:\wamp64\www
2) Create database name nurserydb in phpmyadmin and import the nurserydb.sql in mininursery\db folder
3) Configure the config.php according to your database credentials (change the username and password)
